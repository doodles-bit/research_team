# dungeon-01은 좌절이 아닌 "건강한 벽"

## 인사이트
dungeon-01의 높은 사망률(55~62%)은 유저를 좌절시키는 장벽이 아니라, 도전 동기를 자극하는 건강한 난관으로 기능한다. 연속 실패할수록 재도전율이 올라가고, 사망 후 이탈은 2.6%에 불과하다.

## 근거
- 연속 사망 1회차 재도전율 37.3% → 6회+ 59.3%로 상승 (`ingame_dungeon_exit`, `ingame_dungeon_enter`)
- 사망 후 55%가 5분 내 재진입, 80%가 30분 내 재진입
- 6회 이상 시도한 유저(79명)의 클리어율 94.9%
- 사망 경험 유저 114명 중 이탈 3명(2.6%)
- 미클리어 29명은 평균 2.7회만 시도 후 다른 활동 지속 — 좌절이 아닌 조기 이탈
- 데이터 기간: 2025-12-05 ~ 2025-12-11 (알파테스트)

## 활용 제안
- **난이도 조정 의사결정**: 사망률이 높다는 이유만으로 난이도를 낮출 필요 없음. 행동 데이터가 도전 동기를 보여줌
- **Lv31 = 클리어 가능 전환점**: 사망률 82.1%(Lv27) → 35.6%(Lv31). 던전 가이드 레벨, UI 힌트 기준으로 활용 가능
- **미클리어 유저 개입**: 29명은 시도 자체가 적으므로, 난이도 하향보다 재도전 유도(보상, 가이드)가 더 효과적일 수 있음
