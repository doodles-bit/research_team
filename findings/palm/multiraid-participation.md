# PalM 멀티레이드 — 86.9%가 솔로, "멀티"는 이름뿐

## 인사이트
멀티레이드는 PalM의 유일한 협동 콘텐츠이지만, 전체 세션의 86.9%가 솔로 플레이다. 멀티 참여 시 클리어율이 크게 높지만(R2: 94.3% vs 솔로 69.1%), 대부분의 유저가 솔로로 도전한다. 멀티레이드 참여→잔존 효과는 레벨을 통제하면 소멸하여, 참여 자체보다 플레이 깊이의 부산물이다.

## 근거
- 901명 참여(전체 2,123명의 42.4%), 3,509건 입장, 3,020개 세션 분석 (`ingame_multiraid_enter`/`exit`, 알파 기간)
- 솔로 세션 86.9%(2,623/3,020), 2인 10.2%, 3인+ 2.9%
- Lv20+ 유저의 99.8%가 참여 — 레벨 진행의 자연스러운 결과
- 잔존율: 참여자 75.0% vs 비참여자 17.8% → Lv15-19 통제 시 51.7% vs 46.8% (4.9%p, 유의미하지 않음)
- R2 난이도 분기점: 솔로 사망률 R1 3.2% → R2 28.7% 급증
- R2 레벨 도달 후에도 미진행 유저 160명(34.9%) — 동기/인지 문제 가능성
- [탐색적 분석] 데이터 볼륨 약 3,500건으로 소규모, 하위 그룹 분석은 참고용

## 활용 제안
- **멀티 참여 유도 설계**: 86.9% 솔로는 "협동"이 작동하지 않는다는 신호 — 매칭, 보상, UI 개선 검토
- **R2 난이도 분기점 활용**: 솔로 사망률 급증 구간에서 멀티 추천/매칭 자연 유도
- **R2 미진행 유저 160명**: 레벨은 충분하지만 진입하지 않음 — 인지 부족(UI) vs 동기 부족(보상) 원인 파악 필요
- **일일 입장 제한**: [Estimate] chance_remain 필드 패턴에서 레이드 종류별 3회 제한 추정(게임 설계 문서 확인 필요), 하루 1종만 소화 패턴 91.1% — 제한 완화 시 참여량 증가 가능성
