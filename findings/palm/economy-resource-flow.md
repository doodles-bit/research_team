# PalM 경제/자원 흐름 — 제작 시스템이 "음식 공장"으로 수렴

## 인사이트
PalM의 제작 시스템은 고레벨로 갈수록 음식 제작에 극단적으로 집중된다. 전체 제작의 73.6%가 음식이며, Baked Berries(구운 열매) 단일 레시피가 70.7%를 차지한다. 자원 수집도 Day1에는 필드 80%였으나 Day3부터 기지가 역전하여 Day6에는 기지 69.3%로 전환된다.

## 근거
- 1,097명 유저, 593만 자원 수집 이벤트 + 122,437건 제작 분석 (`ingame_nature_resource_collect`, `ingame_building_resource_collect`, `ingame_craft_complete`, 알파 기간)
- 음식 제작 비율: Lv1-10 17.7% → Lv21-30 66.5% → Lv41+ 93.3%
- Baked Berries 상위 68명(6.2%)이 전체 생산의 85.4% 담당
- 기지/필드 자원 비율: Day1 기지 20.4% → Day3 역전 → Day6 기지 69.3%
- 유황(Sulfur)은 76% 기지 의존, 팰 크리스탈은 필드 전용(139만 수집) → 필드 탐색의 핵심 동기
- [가설 기각] D1 기지 경험 → 잔존 효과는 레벨 통제 시 소멸 (잔존 차이가 플레이 깊이의 결과이지 원인이 아님)

## 활용 제안
- **제작 다양성 확대**: 음식 외 제작의 매력도/필요성 증대 — 현재 무기/방어구/건축 제작이 합쳐서 26.4%에 불과. [Estimate] 음식은 반복 소비(매 세션 소진)되지만 무기/방어구는 일회성 제작(한 번 만들면 지속 사용)이므로 제작 빈도 자체의 구조적 차이가 주요 원인일 가능성 — 제작 비용 문제인지, 반복 소비 구조 차이인지 확인 필요
- **Baked Berries 의존도 분산**: 단일 레시피 70.7% 집중은 시스템 단순화 징후 — 다른 음식의 효용을 높이거나 레시피 다양화 검토
- **팰 크리스탈의 전략적 역할**: 유일한 필드 전용 자원으로 필드 탐색을 유지시키는 핵심 — 밸런스 변경 시 필드 활동 감소 위험
- **Lv21-30 제작 폭발**: 인당 제작 10배 증가 구간 — 기지 시스템 튜토리얼/UX 최적화 집중 포인트
