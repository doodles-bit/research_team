# PalM 팰 포획 행동 패턴 — 다양성이 잔존과 상관, HP 75%+ 포획 시도 UX 개선 필요

## 인사이트
포획 다양성(유니크 팰 종 수)이 높은 유저일수록 잔존율이 높다. Day1에 11종 이상 포획한 유저의 D4+ 잔존율은 82.8%로, 5종 이하(36.1%)의 2.3배다. 또한 HP 75% 이상에서 9,836회 포획 시도가 모두 실패하고 있어 UX 피드백이 필요하다.

## 근거
- 1,944명 유저, 238,696건 포획(성공+실패) 분석 (`ingame_pal_capture`, 알파 기간)
- 포획 다양성별 D4+ 잔존율: 1-10종 36.1%, 11-20종 55.1%, 21-30종 70.1%, 31+ 82.8%
- 플레이 시간/접속일수를 통제해도 다양성-잔존 상관관계 유지 (단, 인과관계 미확정)
- HP 구간별 성공률: 10% 이하 92.8%, 10-25% 73.3%, 25-50% 37.5%, 50-75% 7.9%, 75%+ 0.0%
- HP 75%+ 시도 9,836건(전체의 4.1%) — 전원 실패, 자원 낭비
- BOSS_GrassMammoth: 325회 시도 전원 실패 — 포획 불가 보스인데 안내 없음
- 상위 10종이 전체 포획의 54.3% 차지 (집중 구조)

## 활용 제안
- **포획 다양성 유도**: 다양한 팰 포획과 잔존이 상관 — 도감 보상, 다양성 미션 등으로 넓은 포획 장려 검토
- **HP 피드백 UI**: HP 75%+ 에서 포획 시도 시 "HP를 더 낮추세요" 등의 안내 추가 → 불필요한 스피어 소모 방지
- **포획 불가 대상 안내**: BOSS_GrassMammoth 등 포획 불가 보스에 UI 표시 추가
- **인기 팰 밸런스**: Kitsunebi(16,606회)가 압도적 1위 — 의도된 설계인지, 다른 팰의 매력도 개선이 필요한지 검토
