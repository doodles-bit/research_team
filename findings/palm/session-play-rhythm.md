# PalM 알파테스트 세션 패턴 — Day3-4에서 몰입도 급락

## 인사이트
알파테스트 7일간 잔존 유저의 일일 플레이 시간 중앙값이 Day1 116.5분에서 Day6 40.0분으로 66% 감소한다. 특히 Day3-4에서 급락이 집중되며, 장시간 세션(60분+) 비율이 30.6%→19.0%로 줄고 미세 세션(5분 미만)이 18.1%→28.3%로 증가한다. "집중 몰입"에서 "짧은 확인" 모드로 전환되는 구조적 패턴이다. 시간대별로는 KST 12시(점심)와 20시(저녁) 이중 피크가 관찰되며, 저녁 세션이 오전 대비 42.5% 더 길다.

## 근거
- 2,123명 유저, 17,155개 세션 분석 (30분 비활동 기준 세션 분리, `ingame_login` 외 9종 활동 로그)
- D1 시작 코호트 잔존 유저 기준, 일일 플레이 중앙값: Day1 116.5분 → Day3 68.0분 → Day6 40.0분
- 이탈 유저 첫 세션 중앙값 10.5분 vs 잔존 유저 32.8분 (3.1배 차이)
- 피크 시간대: KST 12시(점심) + 20시(저녁) 이중 피크, 저녁 세션이 오전 대비 +42.5% 길음
- 데이터 기간: 2025-12-05 ~ 2025-12-11

## 활용 제안
- **콘텐츠 동기 부여 설계**: Day3-4 급락 시점에 새로운 콘텐츠/보상을 제공하여 몰입도 유지 검토
- **세션 설계**: 저녁(18-23시) 세션이 가장 길고 집중도 높음 — 핵심 콘텐츠(이벤트, 레이드 등)를 이 시간대에 배치
- **이탈 조기 경보**: 첫 세션 10분 이하 유저를 이탈 리스크 그룹으로 조기 식별 가능 (단, 인과관계는 미확정)
- **알파 한계 인식**: 7일 테스트 특성상 초기 호기심 효과가 크므로, 라이브 환경에서는 감소 폭이 다를 수 있음
