# Copperhead 세션 생명주기 — 참여 실패 12.1%이나 76명(7.5%)에 100% 집중, 최신 빌드에서 개선

## 인사이트
Copperhead 베타 세션 참여(Join) 시도 중 12.1%가 실패하지만, 첫 번째 시도 기준 실패율은 3.5%에 불과하다. 전체 237건 실패가 76명(7.5%)에 100% 집중되어 있으며, 나머지 933명은 실패 경험 0회다. 빌드 버전에 따라 실패율이 0%~34.8%로 극단적 편차를 보이며, 4월 초 빌드에서 0~2.4%로 급감하여 개선 추세가 확인된다. 세션의 77.2%가 솔로이며, 멀티플레이 세션은 미션 진행률이 15.0%p 높지만 인과관계는 불확정이다.

## 근거
- 2,747개 세션, 1,955건 참여 시도 분석 (`cphsessioncreated`, `cphsessionjoined`, 2026-02-04~04-11)
- 참여 실패 237건 중 219건(92.4%)이 "UnknownError" — 원인 미식별
- 첫 시도 실패율 3.5%(40/1,129) vs 4회+ 시도 28.2%(145/513) — 재시도할수록 실패율 급증
- 빌드별 실패율: CL-867138(34.8%), CL-869197(34.8%) vs CL-872531(0%), CL-873373(2.4%)
- 솔로 세션 77.2%(2,121/2,747), 멀티 미션 진행률 42.7% vs 솔로 27.7%

## 활용 제안
- **실패 집중 플레이어 프로파일링**: 76명이 공유하는 특성(네트워크, 클라이언트 사양) 파악 → 타겟 수정
- **빌드 변경 이력 대조**: 실패율 급변 빌드(CL-867138, 869197)의 네트워킹 코드 변경 검토
- **멀티플레이 테스트 충분성**: 4~5인+ 세션이 전체의 1.3%에 불과 — 다인원 테스트 확대 검토
- **재시도 UX**: 한 번 실패 시 재시도 실패 확률이 급증 — 실패 시 세션 재생성 유도 검토
